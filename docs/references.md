# References

The following ITU-R Recommendations and academic papers form the foundation of this simulator:

## ITU-R Recommendations
- **ITU-R P.618-13**: Propagation data and prediction methods for Earth-space systems.
- **ITU-R P.676-12**: Attenuation by atmospheric gases.
- **ITU-R P.837-7**: Characteristics of precipitation for propagation modelling.
- **ITU-R P.838-3**: Specific attenuation model for rain.
- **ITU-R P.839-4**: Rain height model for prediction methods.
- **ITU-R P.1853-2**: Tropospheric attenuation time series synthesis.
- **ITU-R S.1066**: GEO elevation angle and slant range geometry.

## Academic Papers
- **Maseng, T. and Bakken, P.M. (1981)**: *A stochastic dynamic model of rain attenuation*. IEEE Transactions on Communications.

## Standards
- **ETSI EN 302 307**: DVB-S2 standard for Eb/N0 threshold references.
